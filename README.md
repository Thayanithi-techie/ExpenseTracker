# 💰 Expense Tracker – Java Console App

A simple console-based Java app to track your daily expenses with description, amount, and date.

## 🔧 Features
- Add expense
- View expense list
- View total spent
- (Optional) Save/load from file

## 🛠 Tech Stack
- Java (Core)
- OOP, ArrayList, Scanner

## 🏁 Run Instructions

```bash
javac ExpenseTracker.java
java ExpenseTracker
